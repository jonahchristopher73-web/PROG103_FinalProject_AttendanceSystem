# PROG103_FinalProject_AttendanceSystem
## Project Overview
This is a GUI based Python application developed for PROG103 Principles of Structured Programming.  
The system allows teachers to record and manage student attendance digitally, replacing manual registers with a simple and efficient solution.

## Features
- Add student attendance records (Present or Absent)
- Display all saved records in a text area
- Clear all records when needed
- Simple GUI with labels, entry fields, buttons, and output area

## Structured Programming Principles
The program demonstrates:
- Variables and Data Types: Dictionary used to store records
- Decision Structures: if statements validate input
- Iteration: for loop displays records
- Functions: add_record, show_records, clear_records separate logic into modules

## SDG Relevance
This project supports SDG 4 Quality Education by:
- Improving accuracy in attendance tracking
- Saving time for teachers
- Promoting digital literacy in schools

## Requirements
- Python 3.x
- Tkinter (comes pre installed with Python)
